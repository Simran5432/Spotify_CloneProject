# 🎧 Spotify Clone

A Spotify clone built using HTML and CSS that replicates the UI of the original Spotify web player.  
This project focuses on layout design, styling, and responsiveness.

---

## ✨ Features
- 🎵 Clean and modern UI design  
- 📱 Responsive layout  
- 🎧 Music player interface  
- 🎨 Styled using pure CSS  

---

## 🛠️ Tech Stack
- HTML  
- CSS
